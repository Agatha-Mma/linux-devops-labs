# Lab 06 — Users and Group Management in Linux

## Overview

This lab focuses on managing users and groups in Linux. System administrators use these tools to control access to system resources and manage permissions across multiple users.

## Objectives

- Create new users
- Delete users
- Create groups
- Add users to groups
- Switch between users
- View user information

## Key Concepts

Linux is a multi-user operating system. Each user has a unique ID and can belong to one or more groups.

Groups allow administrators to manage permissions for multiple users at once.

## Tools Used

- useradd
- userdel
- groupadd
- usermod
- passwd
- id

## Outcome

By completing this lab, I practiced creating, modifying, and deleting users and groups while managing access control in a Linux environment.
