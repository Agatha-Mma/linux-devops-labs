# Lab 06 — Package Management in Linux

## Overview

This lab focuses on managing software packages in Linux systems. Package managers allow system administrators and DevOps engineers to install, update, remove, and manage software efficiently.

In Ubuntu and Debian-based systems, the Advanced Package Tool (APT) is used for package management.

## Objectives

- Update package lists
- Upgrade installed packages
- Install new software packages
- Remove software packages
- Search for available packages

## Key Concepts

Package managers simplify software installation by handling dependencies and downloading packages from trusted repositories.

APT commands allow administrators to keep systems updated and secure.

## Tools Used

- apt
- apt-get
- apt-cache

## Outcome

By completing this lab, I practiced installing, updating, searching, and removing software packages using the APT package manager in Linux.
