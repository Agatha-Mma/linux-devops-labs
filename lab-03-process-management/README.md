# Lab 03 — Linux Process Management

## Overview

This lab focuses on managing processes in Linux. Processes are running programs on the system, and understanding how to monitor and control them is a fundamental skill for system administrators and DevOps engineers.

## Objectives

- View running processes
- Understand process IDs (PID)
- Start and stop processes
- Manage foreground and background tasks
- Terminate processes safely

## Key Concepts

A **process** is any running program in the Linux operating system.

Each process has:

- PID (Process ID)
- Owner
- CPU usage
- Memory usage
- Running state

System administrators and DevOps engineers frequently monitor and manage processes to ensure systems run efficiently.

## Tools Used

- `ps`
- `top`
- `htop`
- `kill`
- `ping`
- `jobs`
- `fg`
- `bg`

## Outcome

By completing this lab, I practiced monitoring running processes and terminating processes using standard Linux tools used in real-world system administration.
