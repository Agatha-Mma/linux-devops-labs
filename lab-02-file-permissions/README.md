# Lab 02 — Linux File Permissions and Ownership

## Overview

This lab focuses on understanding how Linux controls access to files and directories using permissions and ownership. These concepts are critical in system administration, DevOps, and server security.

## Objectives

- Understand Linux permission types
- Modify permissions using chmod
- Change file ownership using chown
- Understand read, write, and execute permissions
- Practice secure file access control

## Key Concepts

Linux permissions are represented by three groups:

Owner | Group | Others

Each group has three permissions:

r → Read  
w → Write  
x → Execute

Example:

-rwxr-xr--

## Tools Used

Linux Terminal  
chmod  
chown  
ls -l

## Outcome

By completing this lab, I gained hands-on experience managing file permissions and ownership in Linux, which is essential for secure system operations.
